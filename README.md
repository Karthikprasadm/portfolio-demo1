## 🚀 Getting Started

To run the project locally:

```bash
# 1. Clone this repo
git clone https://github.com/Karthikprasadm/portfolio-demo1.git
cd portfolio-demo1

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```
